# Load Tests

Load tests use [k6][1].

## Run Locally

You need to [install k6][2].

```sh
export ACCESS_TOKEN = ...
make run-patient-everything
make report
```

## Results

Please look into the [Load Testing](../../docs/performance/load-testing.md) section of the docs.
